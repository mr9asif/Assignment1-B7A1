# TypeScript & OOP Assignment

## Overview

In this assignment, I solved 7 different problems based on:

- TypeScript
- Object-Oriented Programming (OOP)

The main goal of this assignment was not only solving problems, but also understanding how TypeScript helps write safer and cleaner code.

Along with the problem solving, I also wrote two small blogs based on concepts I learned during the process.

---

# Topics Covered

## TypeScript Concepts

- Types & Interfaces
- Functions
- Type Narrowing
- `any` vs `unknown`
- Generics
- Class & Object
- Error Prevention using TypeScript

---

## OOP Concepts

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

# Blogs Written

Some problems were straightforward, but some parts were confusing at first.

Especially:

- Understanding when to use interfaces vs types
- Handling unknown data safely
- Writing reusable class structures
- Understanding OOP behavior practically

Sometimes the logic was easy, but writing it properly in TypeScript took time.

I also noticed TypeScript forces us to think more carefully before writing code, which actually helps reduce mistakes.
